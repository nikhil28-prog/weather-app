# Weather App (Flask + API)

This is a simple Weather Application built using Python and Flask.
It fetches real-time weather data from an external API and displays temperature and weather conditions for any city.

## Features

* Search weather by city name
* Real-time temperature display
* Weather description
* Clean and styled user interface
* API integration
* Responsive design

## Technologies Used

* Python
* Flask
* Requests library
* HTML
* CSS
* Weather API

## How to Run the Project

1. Install required libraries:

pip install flask requests

2. Run the application:

python weather_app.py

3. Open your browser and go to:

http://127.0.0.1:5000
